# HTML Cheatsheet

A simple HTML cheatsheet built using **HTML, CSS, Prism.js, and Prism.css** to display syntax-highlighted code examples.

## 📌 Overview

This project is a reference guide for common HTML elements and concepts.

## 🚀 Features

- Syntax-highlighted HTML code snippets
- Organized sections for easy navigation
- Lightweight and beginner-friendly
- Code highlighting and copy button powered by custom Prism.css and Prism.js

## 🛠️ Technologies Used

- HTML5
- CSS3
- Prism.js
- Prism.css

## 📂 Project Structure

```
project-folder/
│── index.html
│── style.css
│── prism.css

│── copy.svg
```

## 📚 Purpose

This project was created to:
- Practice HTML and CSS
- Understand code organization
- Learn how to use Prism.js for syntax highlighting
- Build a useful personal reference tool
